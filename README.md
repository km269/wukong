# Wukong

> **记忆优先、编排驱动的 AI Agent 平台** | Go 1.26 | tRPC 生态 | CortexDB 智能记忆

Wukong 是一个本地优先、可扩展的 AI Agent 平台。它的核心哲学是：**Agent 的真正智能不在于一次对话的表现，而在于跨对话的记忆积累、多 Agent 的协同编排、和技能的持续进化**。

```bash
# 安装
go install github.com/km269/wukong/cmd/wukong@latest

# 交互模式
wukong session

# 配置向导
wukong configure
```
